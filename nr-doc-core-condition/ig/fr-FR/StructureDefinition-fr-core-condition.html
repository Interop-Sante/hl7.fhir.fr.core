: FR Core Condition Profile

: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-condition : 2.2.0
Active : FRCoreConditionProfile

FRCoreConditionProfile est un profil utilisé pour décrire un problème du patient (une pathologie par exemple).

Utilisations:

  • Ce Profil n'est utilisé par aucun autre profil dans ce guide d'implémentation

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
