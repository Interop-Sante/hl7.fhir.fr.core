Extension: FR Core Service Type Duration Extension

Official URL: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-service-type-duration Version: 2.2.0-ballot
Active as of 2026-01-28 Computable Name: FRCoreServiceTypeDurationExtension

Cette extension française permet d’associer le type de service avec la durée de ce service.

This French extension allows to associate the type of service with the duration of this service

Context of Use

    Cette extension peut être utilisée sur les éléments suivants

    • ID délément: PractitionerRole
    • ID délément: HealthcareService
    • ID délément: Schedule

Usage info

Utilisations:

  • Utiliser ce Extension: FR Core Healthcare Service Profile, FR Core Practitioner Role and FR Core Schedule Profile
  • Exemples pour ce Extension: Service de cardiologie and Schedule/FRCoreScheduleExample

You can also check for usages in the FHIR IG Statistics
