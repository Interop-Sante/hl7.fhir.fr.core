: FR Core Human Name Profile

: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-human-name : 2.2.0
Active : FRCoreHumanNameProfile

Profilage du type de données HumanName pour prise en compte de la civilté au niveau de l’élément prefix et du titre au niveau de l’élément suffix .French profile of datatype HumanName with constraints on prefix and suffix

Utilisations:

  • Utilise ce/t/te profil de type de données: FR Core Patient Profile, FR Core Practitioner Profile and FR Core Related Person Profile

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
