Extension: FR Core Patient Birth List Given name Extension

URL officielle: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-patient-birth-list-given-name Version: 2.2.0
Active à partir de 2026-06-04 Nom computable: FRCorePatientBirthListGivenName

Prénoms de l’acte de naissance

Context of Use

    Cette extension peut être utilisée sur les éléments suivants

    • ID d'élément: Patient.name

Usage info

Utilisations:

  • Utilise ce/t/te Extension: FR Core Patient Profile

Extension 0..* Extension(4.0.1) Dans le cas d’une identité créée ou modifiée par un appel au téléservice INSi, il s’agit de la liste des prénoms retournée par le téléservice.
