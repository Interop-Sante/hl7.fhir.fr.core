Resource Profile: FR Core MedicationDispense Profile

Official URL: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-medication-dispense Version: 2.2.0
Active as of 2026-06-22 Computable Name: FRCoreMedicationDispenseProfile

FRCoreMedicationDispenseProfile permet de décrire un traitement dispensé avec notamment le médicament dispensé, la quantité et la référence de la prescription.

Utilisations:

  • Ce Profil n'est utilisé par aucun autre profil dans ce guide d'implémentation

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
