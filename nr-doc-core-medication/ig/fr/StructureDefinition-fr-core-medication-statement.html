Profil de ressource: FR Core MedicationStatement Profile

URL officielle: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-medication-statement Version: 2.2.0
Active à partir de 2026-06-29 Nom computable: FRCoreMedicationStatementProfile

FRCoreMedicationStatementProfile permet de décrire les modalités d’administration d’un médicament au patient (médicament déclaré, mode d’administration, quantité, durée et fréquence). Couvre les bilans médicamenteux et conciliations médicamenteuses.

Utilisations:

  • Ce Profil n'est utilisé par aucun autre profil dans ce guide d'implémentation

Vous pouvez également vérifier les usages dans le FHIR IG Statistics

Résumé

Structures

Cette structure fait référence à ces autres structures:

  • FR Core MedicationRequest Profile (https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-medication-request|2.2.0)
  • FR Core MedicationAdministration Profile (https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-medication-administration|2.2.0)
  • FR Core Medication Profile (https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-medication|2.2.0)
  • Condition (http://hl7.org/fhir/StructureDefinition/Condition|4.0.1)
  • Observation (http://hl7.org/fhir/StructureDefinition/Observation|4.0.1)
  • SimpleQuantity (http://hl7.org/fhir/StructureDefinition/SimpleQuantity|4.0.1)

Slices

Cette structure définit les slices suivantes:

  • The element 1 is sliced based on the value of MedicationStatement.dosage.doseAndRate.dose[x]
  • The element 1 is sliced based on the value of MedicationStatement.dosage.doseAndRate.rate[x]
