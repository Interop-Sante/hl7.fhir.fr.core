: FR Core AllergyIntolerance Profile

: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-allergy-intolerance : 2.2.0
Active : FRCoreAllergyIntoleranceProfile

FRCoreAllergyIntoleranceProfile est un profil utilisé pourdécrire une allergie ou une hypersensibilité non allergique ou une intolérance ou une idiosyncrasie.

Utilisations:

  • Ce Profil n'est utilisé par aucun autre profil dans ce guide d'implémentation

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
