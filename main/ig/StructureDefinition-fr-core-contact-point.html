Data Type Profile: FR Core Contact Point Profile

Official URL: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-contact-point Version: 2.2.0-ballot
Active as of 2025-10-15 Computable Name: FRCoreContactPointProfile

Profil du type de données ContactPoint pour la France, ce profil ajoute le type d’e-mail tel que la bal mss. French profile of ContactPoint datatype

Usages:

  • Use this DataType Profile: FR Core Healthcare Service Profile, FR Core Location Profile, FR Core Organization Profile, FR Core Patient Profile... Show 3 more, FR Core Practitioner Role, FR Core Practitioner Profile and FR Core Related Person Profile
