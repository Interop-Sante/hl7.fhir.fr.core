ValueSet: FR Core ValueSet Encounter discharge disposition

Official URL: https://hl7.fr/ig/fhir/core/ValueSet/fr-core-vs-encounter-discharge-disposition Version: 2.0.0
Standards status: Informative Maturity Level: 1 Computable Name: FRCoreValueSetEncounterDischargeDisposition

This value set defines a set of codes that can be used to where the patient left the hospital Circonstances de sortie de l’hôpital.

References

  • FR Core Encounter Profile

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in https://hl7.fr/ig/fhir/core/CodeSystem/fr-core-cs-circonstances-sortie
  • Include all codes defined in http://terminology.hl7.org/CodeSystem/discharge-disposition
