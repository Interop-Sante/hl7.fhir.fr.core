Resource Profile: FR Core Organization Profile

Official URL: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-organization Version: 2.2.0-ballot
Active as of 2026-01-07 Computable Name: FRCoreOrganizationProfile

Profil de la ressource Organization pour la France. Il s’agit du profil de base pour les organisations en France.

Usage

La ressource Organization est utilisée pour représenter une entité organisationnelle, partie de la structure de l’organisation de santé. En particulier, elle est adaptée pour représentée

  • une structure interne
  • un service
  • un pôle d’activité
  • un centre de responsabilité
  • un centre d’activité
  • un département

Ajout du profil FR

Ce profil ajoute par rapport à la ressource du standard

  • differents types d’identifiant utilisés en France
  • la possibilité de définir les motifs de fermeture (closureReason) et d’ouverture (openReason) de l’organisation
  • la possibilité de définir une période d’utilisation de l’organisation
  • la possibilité de définir un nom court pour l’organisation
  • la possibilité de définir une description pour l’organisation

Utilisations:

  • Dérivé de ce Profil: FR Core Organization Etablissement Profile, FR Core Organization UAC Profile and FR Core Organization UF Profile
  • Référer à ce Profil: FR Core Appointment Operator Extension, FR Core Encounter Profile, FR Core Healthcare Service Profile, FR Core Observation Bmi Profile... Show 12 more, FR Core Observation Body Height Profile, FR Core Observation Body Temperature Profile, FR Core Observation Body Weight Profile, FR Core Observation Blood Pressure Profile, FR Core Observation Head Circum Profile, FR Core Observation Heart Rate Profile, FR Core Observation Respiratory Rate Profile, FR Core Observation Oxygen Saturation Profile, FR Core Organization Extension - Membre d'organisation, FR Core Organization Profile, FR Core Patient Profile and FR Core Practitioner Role
  • Exemples pour ce Profil: HOPITAL INTERCOMMUNAL DE LA PRESQU'ILE G, Depertement d'Endocrinologie, Pole Medecines spécialisées, Service Endocrino Diabeto and Service Dietetique
