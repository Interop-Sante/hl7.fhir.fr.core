Extension: FR Core Lunar Date Extension

URL officielle: https://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-lunar-date Version: 2.2.0
Active à partir de 2026-06-29 Nom computable: FRCoreLunarDateExtension

Date de naissance approximative du patient. Approximate birthdate of the patient

Context of Use

    Cette extension peut être utilisée sur les éléments suivants

    • ID d'élément: Patient

Usage info

Utilisations:

  • Exemples pour ce/t/te Extension: Patient/FRCorePatientDeceasedExample

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
